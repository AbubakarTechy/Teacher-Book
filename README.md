   # Teacher-Book.
This is made on Javascript fronted and python backend with flask.
It based on real life problem.

## Description
Teacher-Book is a web application designed to address real-world classroom management challenges.

## Features
- Student attendance tracking
- Grade management
- Assignment submission and tracking
- Real-time notifications
- User-friendly dashboard

## Tech Stack
- **Frontend**: JavaScript
- **Backend**: Python with Flask
- **Database**: SQLite/PostgreSQL
- **Deployment**: Docker

## Installation
1. Clone the repository
2. Install dependencies
3. Run the application

## Usage
Start the Flask server and navigate to the application in your browser.
